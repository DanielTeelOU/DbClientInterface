Connecting your Github repository

Follow the steps below to link Assemble to Github

On Github navigate to the desired repository that you wish to track.

Once there, select the settings panel on the far right of the navigation bar.

After the page loads look to the left of the screen and select the option titled Webhooks.

Connect a webhook by pasting the URL of this page. This will provide the data for this application

Once connected paste the URL into the blank on the main page and select enter.
